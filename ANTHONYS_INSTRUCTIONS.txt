Designed by Anthony — single Next.js app (root) — operator notes
================================================================

Location: /Users/anthonyjones/Web Design/Designed_By_Anthony

Layout
------
• One Next.js 16 app at the repo root: `src/` (marketing + `/lighthouse`), `public/`, `next.config.ts`, `apphosting.yaml`.
• Supporting folders at root (not separate deploys from this repo): `customer-site-embeds/`, `vertaflow-crm/` stubs or assets as applicable — see README.

Global theme (single source of truth)
-------------------------------------
• Author global tokens in: `src/styles/theme.css`
• Keep in sync: `src/design-system/tokens.css` (same :root variables/values).
• App CSS entry: `src/design-system/dba-global.css`

First-time setup
----------------
1. cd to Designed_By_Anthony
2. npm install
3. Copy `.env.example` → `.env` / `.env.local` as needed for local dev.

Development
-----------
From repo root:
  npm run dev     — builds `public/scripts/site.js`, then Next on :3000
  npm run build   — prebuild (site script + sync headers) + production build

Cursor / IDE
------------
Open the folder "Designed_By_Anthony" as the workspace root.

Firebase App Hosting
--------------------
• Root Directory: repository root (`.`).
• Install / build: `npm ci` and `npm run build` (see `package.json`; `apphosting.yaml` at repo root configures the Cloud Run backend).
• Connect the **GitHub** repo in Firebase console; set `LEAD_WEBHOOK_URL` and other env in App Hosting → Environment / Secrets.

Cloudflare / GCP
-----------------
Follow your platform blueprint; env vars are scoped per backend.

VertaFlow CRM (separate product)
--------------------------------
• Admin and accounts subdomains redirect to **vertaflow.io** from `src/proxy.ts`. This marketing app does not ship the CRM codebase.

Public leads
------------
• Marketing forms POST to **`/api/contact`**, which forwards JSON to **`LEAD_WEBHOOK_URL`** (e.g. Convex → Slack). Env: `src/lib/env/marketing.ts`, routes under `src/app/api/`.
• Lead payload contract: `src/lib/lead-form-contract.ts`
